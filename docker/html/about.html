The IRIS Translation System was developed to translate text from English into Irish and v.v. The Translation System allows to upload English and/or Irish monolingual/parallel data. When the data is uploaded, the system evaluates the data and starts to retrain the translation models (phrase table/language model). After the training process is done, the new data is accessible through the web interface. Additional to the IRIS web interface, a RESTful service is available, which enables the access of IRIS with other applications. For more information visit this website. The evaluation set (testset 1) currently used in IRIS: iris_eval_set.tgz


The Iris System was presented at the LREC'16 conferece in Portoroz, Slovenia. If you use any parts of IRIS in your own work, please cite the following publication:
